## Installation

```bash
  npm install --save-dev webpack
  # or specific version
  npm install --save-dev webpack@<version>
  +
  npm install --save-dev @babel/preset-react

```

## Usage

```bash
npm run w-build or yarn w-build
+
npm run serve or yarn serve

```

## Tech Stack

**Client:** React, Css

**Environnement:** Babel, Webpack
