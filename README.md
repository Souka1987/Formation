## Installation

```bash
  # React
  npx create-react-app my-app
  cd my-app
  +
  # NPM
  npm install redux

  # Yarn
  yarn add redux

  # Json-server
  npm i -g json-server
```


## Usage

```
npm start or yarn start
+
run db :
npm run server or yarn server
```
